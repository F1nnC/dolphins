## Multiple of 3
1. Set number to input
2. If number MOD 3 = 0, display "Multiple of 3"
3. Else, display "Not a multiple of 3".

number ← input
IF (number MOD 3 = 0)
{
    DISPLAY("Multiple of 3")
}
ELSE
{
    DISPLAY ("Not a multiple of 3")
}
